Plan · Drei Phasen, ~2 Personentage

Phase 2 · ConPty Reflow

~2h · Bonus

  • Rust: get_windows_build()
  • Frontend: dynamic windowsPty.buildNumber
  • Win11 (≥21376) → Reflow ON
  • Win10 22H2 (19045) → no-op (wie zuvor)

Akzeptanz: Window schmaler ziehen → alte Zeilen wrapen sauber neu (Win11).

Phase 3 · Persist über Restart

~1 PT · separater PR

  • @xterm/addon-serialize@0.14.0
  • Storage: ${app_data}/scrollback/<id>.vt.gz
  • Save: on-close + 60s throttled
  • Restore: vor PTY-Spawn
  • GC: removeSession → delete file

Akzeptanz: App schließen → öffnen → Resume → kompletter Verlauf da.
